# RISK MATRIX

## 使用方法

在頁面 URL 上加上以下參數：

- `data`: 表格資料 或  `url`: JSON 檔案的網址

- `lang`: 顯示語言，支援 `zh`（中文）、`en`（英文）（預設值）、`ja`（日文）、`ko`（韓文）
